Associating resources
#####################


.. toctree::
   :maxdepth: 2

   linking-data.rst
   linking-thumbnail.rst

GeoNetwork allows to link resources to each other using the linkage panel. Metadata records can be associated to each other as parent-child, series-dataset, or service-dataset. To each metadata document resources like datasets, documents, thumbnails can be attached.
